Projects

Seminar hall booking System

A web-based application developed to manage and simplify the booking process of a seminar hall within the institution. The system allows users to check availability and reserve a single seminar hall for specific dates and time slots. It prevents double bookings by validating time conflicts and stores reservation details in a database. Built using Python (Flask) for backend, HTML, CSS, JavaScript for frontend, and MySQL (XAMPP) for database management, the system ensures efficient scheduling and easy access for users and administrators.

Realtime Scam site detectiont

A machine learning-based web application designed to detect whether a given URL is safe or phishing (scam) in real time. The system analyzes URL patterns and extracts features such as length, presence of special characters, suspicious keywords, and domain structure to identify malicious behavior. It uses algorithms like Logistic Regression and XGBoost trained on datasets collected from sources like Kaggle. When a user inputs a URL, the system processes it, applies feature extraction, and predicts the risk level (Low, Medium, High) along with a probability score.
